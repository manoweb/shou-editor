Layer System

Integration

Drop in.
Start building.

No npm, no webpack, no build step. Include two script files and initialize with a single line. Works with any framework or plain HTML.

  • Works with file:// protocol
  • No CORS issues with local images
  • JSON configuration for everything
  • Full programmatic API
  • Embeddable or modal mode
  • Custom save endpoints via POST
const editor = ShouEditor.init('#editor', {
  theme: 'dark',
  width: '100%',
  height: '100vh',
  saveTarget: 'remote',
  saveEndpoint: 'https://api.example.com/save',
  onSaveSuccess: (res) => console.log('Saved!', res)
});

// Get the generated code
const { html, css, js } = editor.getCode();
// Embedded mode
const imgEditor = JSImageEditor.init('#container', {
  theme: 'dark',
  lang: 'en',
  preset: { width: 800, height: 600 },
  onSave: (base64) => uploadImage(base64)
});

// Or open as a modal
JSImageEditor.open(existingImageUrl, {
  onSave: (base64) => updateAvatar(base64)
});
// Bind editor output to a form
editor.bindToForm('#my-form');
// Creates hidden fields: editor_html, editor_css, editor_js

// Or use a single JSON field
editor.bindToForm('#my-form', {
  field: 'page_content'
});

// Custom field names
editor.bindToForm('#my-form', {
  htmlField: 'body_html',
  cssField: 'body_css',
  jsField: 'body_js'
});
