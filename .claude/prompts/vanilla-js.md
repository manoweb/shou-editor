# Prompt: Desarrollo Vanilla JavaScript

Cuando escribas código para este proyecto:

1. **JavaScript**
   - Usa ES6+ (const, let, arrow functions, template literals, destructuring)
   - Manipulación DOM nativa (querySelector, addEventListener, etc.)
   - Fetch API para peticiones HTTP
   - Async/await para código asíncrono
   - Módulos ES6 nativos cuando sea necesario

2. **HTML**
   - HTML5 semántico
   - Atributos data-* para datos personalizados
   - Accesibilidad (ARIA cuando sea necesario)

3. **CSS**
   - CSS3 moderno
   - Variables CSS (custom properties)
   - Flexbox y Grid para layouts
   - Sin preprocesadores

4. **Prohibido**
   - Frameworks JS
   - Librerías externas
   - Compiladores/bundlers
   - CDN de terceros
