# Skills del Editor JavaScript

## /preview
Abre una vista previa del código HTML/CSS/JS actual en el navegador.

## /validate
Valida la sintaxis del código JavaScript, HTML y CSS.

## /format
Formatea el código siguiendo las convenciones del proyecto.

## /export
Exporta el proyecto como archivo HTML único con CSS y JS embebido.

## /minify
Minifica el código para producción (sin dependencias externas).
